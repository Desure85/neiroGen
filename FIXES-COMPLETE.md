## 🐛 Проблемы которые были

### 1. Список, чекбокс и таблица рендерились как плейсхолдеры
**Причина:** Не был реализован рендеринг для этих типов элементов  
**Симптом:** Показывалось "Укажите URL изображения"

### 2. Круг, эллипс, звезда прыгают при перетаскивании
**Причина:** Неправильные координаты для центрированных фигур  
**Симптом:** При drag элемент смещался назад

### 3. Линии редактируются прямоугольником
**Причина:** Использовался стандартный Transformer  
**Симптом:** Нельзя было редактировать начало и конец отдельно

## ✅ Решения

### 1. Рендеринг list, checkbox, table

#### Список
```typescript
if (element.type === 'list') {
  const items = element.items || ['Пункт 1', 'Пункт 2', 'Пункт 3']
  const listStyle = element.listStyle || 'numbered'
  
  return (
    <KonvaGroup>
      <KonvaRect /> // Фон
      {items.map((item, index) => (
        <KonvaText 
          text={
            listStyle === 'numbered' ? `${index + 1}. ${item}` :
            listStyle === 'bulleted' ? `• ${item}` :
            `☐ ${item}`
          }
        />
      ))}
    </KonvaGroup>
  )
}
```

**Поддерживаются стили:**
- `numbered` → 1. 2. 3.
- `bulleted` → • • •
- `checkbox` → ☐ ☐ ☐

#### Чекбокс
```typescript
if (element.type === 'checkbox') {
  return (
    <KonvaGroup>
      <KonvaRect cornerRadius={4} /> // Квадрат
      <KonvaLine points={[...]} />    // Галочка ✓
    </KonvaGroup>
  )
}
```

**Визуал:**
- Квадрат с закругленными углами
- Зелёная галочка внутри

#### Таблица
```typescript
if (element.type === 'table') {
  const rows = 3, cols = 3
  
  return (
    <KonvaGroup>
      <KonvaRect />  // Внешняя рамка
      {/* Горизонтальные линии */}
      {Array.from({ length: rows - 1 }).map(...)}
      {/* Вертикальные линии */}
      {Array.from({ length: cols - 1 }).map(...)}
    </KonvaGroup>
  )
}
```

**Визуал:**
- Сетка 3×3
- Толстая внешняя рамка
- Тонкие внутренние линии

### 2. Исправление drag центрированных фигур

**Было:**
```typescript
<KonvaCircle
  x={element.x + element.width / 2}  // ← Неправильно!
  y={element.y + element.height / 2}
  radius={radius}
/>
```

**Проблема:** 
- `element.x` и `element.y` - это координаты левого верхнего угла bounding box
- Мы рендерили фигуру со смещением
- При drag Konva обновлял координаты центра фигуры
- Мы записывали их обратно как координаты левого верхнего угла
- **Результат:** прыжок!

**Стало:**
```typescript
<KonvaCircle
  x={element.x}                    // ← Правильно!
  y={element.y}
  offsetX={-element.width / 2}     // ← Смещение!
  offsetY={-element.height / 2}
  radius={radius}
/>
```

**Решение:**
- Используем `element.x/y` напрямую
- Добавляем `offsetX/offsetY` для центрирования
- Теперь координаты согласованы!

**Применено к:**
- ✅ Круг (`circle`)
- ✅ Эллипс (`ellipse`)
- ✅ Звезда (`star`)
- ✅ Шестиугольник (`hexagon`)

### 3. Редактирование линий по точкам

**Было:**
```
Выделена линия
┌─────────────────┐
│  ────────────>  │  ← Прямоугольник с якорями
└─────────────────┘
```

**Стало:**
```
Выделена линия
●──────────────>●  ← Две точки: начало и конец
```

**Реализация:**

```typescript
{/* Кастомные точки для линий */}
{selectedElement && selectedElement.type === 'line' && (
  <>
    {/* Начальная точка */}
    <KonvaCircle
      x={selectedElement.x}
      y={selectedElement.y}
      radius={6}
      fill="#2563eb"
      draggable
      onDragMove={(e) => {
        updateElement(selectedElement.id, {
          x: e.target.x(),
          y: e.target.y(),
        })
      }}
    />
    
    {/* Конечная точка */}
    <KonvaCircle
      x={selectedElement.x2}
      y={selectedElement.y2}
      radius={6}
      fill="#2563eb"
      draggable
      onDragMove={(e) => {
        updateElement(selectedElement.id, {
          x2: e.target.x(),
          y2: e.target.y(),
        })
      }}
    />
  </>
)}

{/* Transformer только для НЕ-линий */}
{selectedElement && selectedElement.type !== 'line' && (
  <KonvaTransformer ... />
)}
```

**Визуал:**
- Синие кружки (radius: 6px)
- Белая обводка
- Перетаскиваемые независимо
- Как в Flameshot! 🎯

**Поведение:**
1. Выделяешь линию → появляются две точки
2. Тянешь начало → линия двигается от начала
3. Тянешь конец → линия двигается от конца
4. Можно наклонить линию как угодно!

## 🎨 Использование

### Список
1. Нажать **[≡ Список]**
2. Появится список с 3 пунктами
3. В настройках выбрать стиль:
   - Нумерованный
   - Маркированный
   - Чекбокс

### Чекбокс
1. Нажать **[☑ Чекбокс]**
2. Появится квадрат с галочкой
3. Растянуть до нужного размера

### Таблица
1. Нажать **[⊞ Таблица]**
2. Появится сетка 3×3
3. Растянуть до нужного размера

### Линия по точкам
1. Нажать **[— Линия]**
2. Выделить линию
3. Появятся **две синие точки**: ● начало и ● конец
4. Перетащить любую точку куда нужно!

## 💡 Технические детали

### offsetX/offsetY
```typescript
// Для прямоугольников (x,y = левый верхний угол):
x={element.x}
y={element.y}

// Для центрированных (x,y = центр):
x={element.x}
y={element.y}
offsetX={-element.width / 2}   // Смещаем назад влево
offsetY={-element.height / 2}  // Смещаем назад вверх
```

### Кастомные точки vs Transformer
```typescript
// Transformer - для фигур (прямоугольник с якорями)
{selectedElement.type !== 'line' && (
  <KonvaTransformer />
)}

// Кастомные точки - для линий (две точки)
{selectedElement.type === 'line' && (
  <>
    <KonvaCircle /> // Начало
    <KonvaCircle /> // Конец
  </>
)}
```

### Рендеринг списка
```typescript
items.map((item, index) => (
  <KonvaText
    text={
      listStyle === 'numbered' ? `${index + 1}. ${item}` :
      listStyle === 'bulleted' ? `• ${item}` :
      `☐ ${item}`
    }
    y={10 + index * 24}  // lineHeight
  />
))
```

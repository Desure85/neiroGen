## 🔧 Рекомендации по запуску

### Для запуска тестов требуется:
1. Установить Node.js и npm
2. Выполнить `npm install` в директории frontend
3. Установить Playwright: `npm run playwright:install`
4. Запустить бэкенд сервер
5. Выполнить: `npm run test:e2e`

### Переменные окружения:
```bash
E2E_ADMIN_EMAIL=admin@demo.local
E2E_ADMIN_PASS=password
BASE_URL=http://127.0.0.1:3001
```

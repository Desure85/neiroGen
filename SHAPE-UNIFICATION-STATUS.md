# 🔄 Объединение фигур в прогрессе

**Статус:** В процессе реализации  
**Дата:** 5 ноября 2025

---

## ✅ Что сделано

### 1. Типы обновлены
- ✅ Заменен `'circle' | 'rectangle'` на единый `'shape'`
- ✅ Добавлено свойство `shapeType: 'circle' | 'rectangle' | 'ellipse' | ...`
- ✅ Убрано свойство `radius`, теперь используем `width/height`

### 2. UI кнопки обновлены
- ✅ Кнопки разбиты на группы:
  - **Инструменты:** Линия, Стрелка, Номер
  - **Фигуры:** Круг, Прямоугольник, Эллипс
  - **Дополнительно:** Список, Чекбокс, Таблица

### 3. Логика создания
- ✅ Функция `addElement` принимает опциональный `shapeType`
- ✅ Кнопки передают: `addElement('shape', 'circle')`

## 🔧 Что нужно доделать

### 1. Рендеринг фигур
Заменить старые проверки:
```typescript
// Было:
if (element.type === 'circle') { ... }
if (element.type === 'rectangle') { ... }

// Должно быть:
if (element.type === 'shape') {
  if (element.shapeType === 'circle') { ... }
  if (element.shapeType === 'rectangle') { ... }
  if (element.shapeType === 'ellipse') { ... }
}
```

### 2. Панель свойств
Заменить проверки для редактирования:
```typescript
// Было:
if (selectedElement.type === 'circle') { ... }
if (selectedElement.type === 'rectangle') { ... }

// Должно быть:
if (selectedElement.type === 'shape') {
  // Общие контролы
  // + dropdown для выбора shapeType
}
```

### 3. Растягивание круга
```typescript
// Круг теперь использует width/height вместо radius
if (element.shapeType === 'circle') {
  const radius = Math.min(element.width, element.height) / 2
  return <KonvaCircle radius={radius} />
}

// Эллипс использует разные радиусы
if (element.shapeType === 'ellipse') {
  return <KonvaEllipse 
    radiusX={element.width / 2}
    radiusY={element.height / 2}
  />
}
```

## 💡 Преимущества объединения

### Было (раздельно):
```
[○ Круг] - не растягивается, только радиус
[□ Прямоугольник] - растягивается
```

### Стало (объединено):
```
[○ Круг] - растягивается как прямоугольник!
[□ Прямоугольник] - растягивается
[○ Эллипс] - растягивается по-разному в X и Y
```

**Единый интерфейс:**
- Все фигуры растягиваются одинаково
- Выбор формы в dropdown
- Общие свойства (обводка, заливка, скругление)

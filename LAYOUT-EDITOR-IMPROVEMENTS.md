## 📝 Что исправлено

### 1. Дублирование настроек цветов

**Было:**
```tsx
// Для placeholder - свои контролы
{selectedElement.type === 'placeholder' && (...)}

// Для instructions - другие контролы
{selectedElement.type === 'instructions' && (...)}
```

**Стало:**
```tsx
// Единый блок для всех элементов с рамками
{(selectedElement.type === 'instructions' || 
  selectedElement.type === 'placeholder' || ...) && (
  <div>Оформление</div>
)}
```

### 2. Нефункциональный scaleContent

**Было:**
```tsx
<input 
  type="checkbox"
  checked={selectedElement.scaleContent !== false}
  onChange={...}
/>
<label>Масштабировать содержимое</label>
```

**Проблема:** Чекбокс был, но scaleContent нигде не использовался при рендеринге!

**Решение:** Удалили за ненадобностью (контент и так корректно отображается)

## 📐 Номер

### Настройки

```
┌─────────────────────────┐
│ Форма контура          │
│ [Круг              ▼]  │ ← Dropdown!
│ Номер        [1    ▼]  │
│ Размер шрифта [24  ▼]  │
│ Цвет текста    [⬛]    │
│ Цвет контура   [⬛]    │ ← Только если есть контур
└─────────────────────────┘
```

### Формы контура

1. **Круг** ○
   - Классический вариант
   - Круглая форма

2. **Квадрат** □
   - Прямоугольная форма
   - Чёткие углы

3. **Треугольник** △
   - Треугольная форма
   - Острые углы

4. **Звезда** ★
   - Звезда с 5 концами
   - Привлекает внимание

5. **Без контура**
   - Только номер без обрамления
   - Минималистичный стиль

### Рендеринг

```typescript
{numberShape === 'circle' && <KonvaCircle />}
{numberShape === 'square' && <KonvaRect />}
{numberShape === 'triangle' && <KonvaLine closed />}
{numberShape === 'star' && <KonvaStar />}
{numberShape === 'none' && null}

<KonvaText>{numberValue}</KonvaText>
```

## ≡ Список

### Настройки

```
┌─────────────────────────────────────┐
│ Стиль списка                        │
│ [Нумерованный (1. 2. 3.) ▼]        │ ← Dropdown!
│ Цвет обводки          [⬛]          │
│ Цвет фона             [⬜]          │
└─────────────────────────────────────┘
```

### Стили списка

1. **Нумерованный (1. 2. 3.)**
   ```
   1. Пункт 1
   2. Пункт 2
   3. Пункт 3
   ```

2. **Маркированный (• • •)**
   ```
   • Пункт 1
   • Пункт 2
   • Пункт 3
   ```

3. **Чекбоксы (☐ ☐ ☐)**
   ```
   ☐ Пункт 1
   ☐ Пункт 2
   ☐ Пункт 3
   ```

### Рендеринг

```typescript
items.map((item, index) => (
  <KonvaText
    text={
      listStyle === 'numbered' ? `${index + 1}. ${item}` :
      listStyle === 'bulleted' ? `• ${item}` :
      `☐ ${item}`
    }
  />
))
```

## ⊞ Таблица

### Настройки

```
┌─────────────────────────┐
│ Количество строк [3 ▼] │
│ Количество столбцов [3▼]│
│ Цвет линий     [⬛]    │
└─────────────────────────┘
```

### Параметры

- **Строки:** 1-10
- **Столбцы:** 1-10

### Примеры

**2×2:**
```
┌───┬───┐
│   │   │
├───┼───┤
│   │   │
└───┴───┘
```

**3×4:**
```
┌───┬───┬───┬───┐
│   │   │   │   │
├───┼───┼───┼───┤
│   │   │   │   │
├───┼───┼───┼───┤
│   │   │   │   │
└───┴───┴───┴───┘
```

### Рендеринг

```typescript
const rows = element.tableRows || 3
const cols = element.tableCols || 3

// Внешняя рамка (толстая)
<KonvaRect strokeWidth={2} />

// Горизонтальные линии (тонкие)
{Array.from({ length: rows - 1 }).map(...)}

// Вертикальные линии (тонкие)
{Array.from({ length: cols - 1 }).map(...)}
```

## 💡 Использование

### Номер

**Пример: Шаги упражнения**

1. Добавить номер [# Номер]
2. В настройках выбрать форму: **Круг**
3. Установить номер: **1**
4. Цвет текста: черный
5. Цвет контура: синий
6. Дублировать для номеров 2, 3, 4...

**Результат:**
```
Ⓐ Шаг 1   Ⓑ Шаг 2   Ⓒ Шаг 3
```

**Без контура:**
- Выбрать форму: **Без контура**
- Получится просто номер: `1` `2` `3`

### Список

**Пример: Инструкция для упражнения**

1. Добавить список [≡ Список]
2. Выбрать стиль: **Нумерованный**
3. Настроить цвета

**Результат:**
```
┌──────────────────────────┐
│ 1. Карандаш             │
│ 2. 9 шагов              │
│ 3. 6 шагов              │
└──────────────────────────┘
```

### Чекбокс

**Пример: Задание с выбором**

1. Добавить чекбокс [☑ Чекбокс]
2. Поставить галочку (отмечен)
3. Разместить рядом с правильным ответом

**Результат:**
```
САНКИ  ☑
ШАНКИ  ☐
```

### Таблица

**Пример: Сравнительная таблица**

1. Добавить таблицу [⊞ Таблица]
2. Установить: 3 строки, 2 столбца
3. Цвет линий: черный

**Результат:**
```
┌────────┬────────┐
│   С    │   Ш    │
├────────┼────────┤
│ САНКИ  │ ШАНКА  │
├────────┼────────┤
│ СУМКА  │ ШУМКА  │
└────────┴────────┘
```

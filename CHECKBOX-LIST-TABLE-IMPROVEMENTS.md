## 🎯 Что добавлено

### 1. Символы для чекбокса

Теперь чекбокс может отображать разные символы отметки:

**Варианты:**
- ✓ **Галочка** (зелёная) — классический вариант
- ✕ **Крестик** (красный) — для отрицания или ошибки
- • **Точка** (синяя) — минималистичный вариант

**Настройка:**
```
┌─────────────────────────┐
│ [✓] Отмечен            │
│ Символ отметки         │
│ [Галочка ✓      ▼]     │
│ Цвет обводки   [⬛]    │
│ Цвет фона      [⬜]    │
└─────────────────────────┘
```

**Рендеринг:**
```typescript
{isChecked && symbol === 'check' && (
  <KonvaLine points={[...]} stroke="#22c55e" />  // Галочка
)}
{isChecked && symbol === 'cross' && (
  <>
    <KonvaLine points={[...]} stroke="#ef4444" />  // \
    <KonvaLine points={[...]} stroke="#ef4444" />  // /
  </>
)}
{isChecked && symbol === 'dot' && (
  <KonvaCircle fill="#3b82f6" />  // Точка
)}
```

### 2. Редактирование элементов списка

Теперь можно редактировать содержимое списка прямо в панели свойств:

**Функции:**
- ✍️ Редактировать текст каждого пункта
- ➕ Добавить новый пункт
- ❌ Удалить пункт

**UI:**
```
┌─────────────────────────────────┐
│ Стиль списка                    │
│ [Нумерованный (1. 2. 3.) ▼]    │
│                                 │
│ Элементы списка:                │
│ ┌─────────────────────┐ [X]    │
│ │ Пункт 1             │        │
│ └─────────────────────┘        │
│ ┌─────────────────────┐ [X]    │
│ │ Пункт 2             │        │
│ └─────────────────────┘        │
│ ┌─────────────────────┐ [X]    │
│ │ Пункт 3             │        │
│ └─────────────────────┘        │
│                                 │
│ [+ Добавить]                    │
│                                 │
│ Цвет обводки    [⬛]           │
│ Цвет фона       [⬜]           │
└─────────────────────────────────┘
```

**Реализация:**
```tsx
{(selectedElement.items || ['Пункт 1', 'Пункт 2', 'Пункт 3']).map((item, index) => (
  <div className="flex gap-1">
    <Input
      value={item}
      onChange={(e) => {
        const newItems = [...(selectedElement.items || [])]
        newItems[index] = e.target.value
        updateElement(selectedElement.id, { items: newItems })
      }}
    />
    <Button onClick={() => {
      const newItems = items.filter((_, i) => i !== index)
      updateElement(selectedElement.id, { items: newItems })
    }}>
      <X />
    </Button>
  </div>
))}

<Button onClick={() => {
  const newItems = [...items, `Пункт ${items.length + 1}`]
  updateElement(selectedElement.id, { items: newItems })
}}>
  <Plus /> Добавить
</Button>
```

### 4. Исправлена правая панель

**Было:**
```
Правая панель растягивалась на всю высоту,
даже когда ничего не выбрано
```

**Стало:**
```
┌─────────────────────────┐
│         [+]             │
│  Выберите элемент       │
│  на макете, чтобы       │
│  изменить его параметры │
└─────────────────────────┘
```

**Компактный плейсхолдер** не занимает лишнее пространство.

**Изменение:**
```tsx
// Было:
<div className="flex h-full flex-col ...">  // ← h-full

// Стало:
<div className="flex flex-col items-center p-6 ...">  // ← без h-full
```

## 💡 Примеры использования

### Чекбокс: Выбор правильного ответа

**Задание:** Найти правильное слово

```
САНКИ  ✓  ← галочка (правильно)
ШАНКИ  ✕  ← крестик (неправильно)
ЗАНКИ  •  ← точка (не знаю)
```

**Настройка:**
1. Добавить 3 чекбокса
2. Первый: символ = галочка, отмечен
3. Второй: символ = крестик, отмечен
4. Третий: символ = точка, отмечен

### Список: Инструкция упражнения

**Задание:** Шаги выполнения

```
Инструкция:
1. Карандаш
2. 9 шагов
3. 6 шагов
4. 5 шагов
```

**Настройка:**
1. Добавить список
2. Стиль: Нумерованный
3. Редактировать пункты:
   - Пункт 1 → "Карандаш"
   - Пункт 2 → "9 шагов"
   - Пункт 3 → "6 шагов"
4. Добавить пункт → "5 шагов"

### Таблица: Сравнение слов

**Задание:** С или Ш?

```
┌──────┬──────┬──────┐

├──────┼──────┼──────┤
│  1   │ СУМКА│ШУМКА │
├──────┼──────┼──────┤
│  2   │ САНКИ│ШАНКА │
└──────┴──────┴──────┘
```

**Настройка:**
1. Добавить таблицу
2. Строки: 3
3. Столбцы: 3
4. (В будущем можно будет редактировать содержимое ячеек)

## 🎨 Цветовая схема символов

### Галочка ✓
- Цвет: `#22c55e` (зелёный)
- Значение: **Правильно**, **Да**, **Выполнено**

### Крестик ✕
- Цвет: `#ef4444` (красный)
- Значение: **Неправильно**, **Нет**, **Ошибка**

### Точка •
- Цвет: `#3b82f6` (синий)
- Значение: **Нейтрально**, **Не знаю**, **В процессе**

## 🔧 Технические детали

### Типы

```typescript
interface CanvasElement {
  // Чекбокс
  checkboxChecked?: boolean
  checkboxSymbol?: 'check' | 'cross' | 'dot'
  
  // Список
  items?: string[]
  listStyle?: 'numbered' | 'bulleted' | 'checkbox'
  
  // Таблица
  tableRows?: number
  tableCols?: number
  tableCells?: string[][]  // Для будущего редактирования
}
```

### Рендеринг чекбокса

```typescript
const symbol = element.checkboxSymbol || 'check'

// Галочка
{symbol === 'check' && (
  <KonvaLine
    points={[
      size * 0.2, size * 0.5,
      size * 0.4, size * 0.7,
      size * 0.8, size * 0.3
    ]}
    stroke="#22c55e"
    strokeWidth={3}
  />
)}

// Крестик
{symbol === 'cross' && (
  <>
    <KonvaLine points={[0.2, 0.2, 0.8, 0.8]} />  // \
    <KonvaLine points={[0.8, 0.2, 0.2, 0.8]} />  // /
  </>
)}

// Точка
{symbol === 'dot' && (
  <KonvaCircle
    x={size / 2}
    y={size / 2}
    radius={size * 0.2}
    fill="#3b82f6"
  />
)}
```

### Редактирование списка

```typescript
// Обновить элемент
const newItems = [...selectedElement.items]
newItems[index] = newValue
updateElement(id, { items: newItems })

// Удалить элемент
const newItems = items.filter((_, i) => i !== index)
updateElement(id, { items: newItems })

// Добавить элемент
const newItems = [...items, `Пункт ${items.length + 1}`]
updateElement(id, { items: newItems })
```

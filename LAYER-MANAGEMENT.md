## 🎯 Проблема

При работе с элементами с непрозрачной заливкой последний добавленный элемент перекрывает предыдущие. Не было возможности изменить порядок отрисовки (z-index).

---

## ✅ Решение

Добавлено управление слоями с 4 кнопками для изменения порядка элементов.

---

## 🔧 Как работает

### Порядок отрисовки в Konva

В Konva элементы рисуются в порядке массива:
```typescript
elements = [
  element1,  // ← Рисуется первым (внизу, задний план)
  element2,  // ← Рисуется вторым
  element3,  // ← Рисуется последним (сверху, передний план)
]
```

**Чем позже элемент в массиве → тем выше он на экране**

## 🎮 Функции управления

### 1. На передний план (⇈)
```typescript
moveToFront(id) {
  // Удаляет элемент из текущей позиции
  // Добавляет в конец массива
  elements.push(element)
}
```

**Эффект:** Элемент становится самым верхним

### 2. Вперёд (↑)
```typescript
moveForward(id) {
  // Меняет местами с элементом выше
  swap(elements[i], elements[i + 1])
}
```

**Эффект:** Элемент поднимается на один слой вверх

### 3. Назад (↓)
```typescript
moveBackward(id) {
  // Меняет местами с элементом ниже
  swap(elements[i], elements[i - 1])
}
```

**Эффект:** Элемент опускается на один слой вниз

### 4. На задний план (⇊)
```typescript
moveToBack(id) {
  // Удаляет элемент из текущей позиции
  // Добавляет в начало массива
  elements.unshift(element)
}
```

**Эффект:** Элемент становится самым нижним

## 🎨 UI

### Панель управления слоями

```
┌─────────────────────────────────┐
│ Порядок слоёв                   │
├─────────────────────────────────┤
│ [⇈]  [↑]  [↓]  [⇊]             │
│  На    Вперёд  Назад   На       │
│ передний              задний    │
│  план                  план     │
└─────────────────────────────────┘
```

**Расположение:** В панели свойств выбранного элемента, под заголовком

**Кнопки:**
- **⇈ (ChevronsUp)** — На передний план
- **↑ (ArrowUp)** — Вперёд на один слой
- **↓ (ArrowDown)** — Назад на один слой
- **⇊ (ChevronsDown)** — На задний план

**Tooltips:** При наведении показывается подсказка

## 💡 Использование

### Пример 1: Фон позади текста

**Проблема:**
```
Добавили прямоугольник с заливкой
Добавили текст
→ Текст перекрыт прямоугольником ❌
```

**Решение:**
1. Выделить прямоугольник
2. Нажать **⇊ На задний план**
3. Теперь текст виден! ✅

### Пример 2: Порядок наложения фигур

**Задача:** Создать рисунок из нескольких фигур

**Действия:**
1. Добавить круг (синий)
2. Добавить квадрат (красный)
3. Добавить треугольник (жёлтый)

**По умолчанию:**
```
Треугольник (сверху)
Квадрат
Круг (снизу)
```

**Изменение порядка:**
- Выделить круг → **⇈ На передний план**
```
Круг (сверху)
Треугольник
Квадрат (снизу)
```

- Выделить квадрат → **↑ Вперёд**
```
Круг (сверху)
Квадрат
Треугольник (снизу)
```

### Пример 3: Выделение важного элемента

**Задача:** Обвести правильный ответ кругом

**Проблема:**
```
Добавили текст "САНКИ"
Добавили круг вокруг
→ Круг перекрывает текст ❌
```

**Решение:**
1. Выделить круг
2. Нажать **⇊ На задний план**
3. Круг теперь позади текста ✅

## 🔍 Технические детали

### Оптимизация

Каждая операция:
1. Создаёт новый массив элементов
2. Обновляет snapshot с задержкой (1 секунда)
3. Отправляет изменения с debounce (500ms)

**Результат:** Плавная работа без лагов

### Проверки границ

```typescript
moveToFront(id) {
  if (index === elements.length - 1) return  // Уже сверху
  // ...
}

moveForward(id) {
  if (index === elements.length - 1) return  // Уже сверху
  // ...
}

moveBackward(id) {
  if (index === 0) return  // Уже снизу
  // ...
}

moveToBack(id) {
  if (index === 0) return  // Уже снизу
  // ...
}
```

**Результат:** Кнопки не ломают состояние

## 📦 Реализация

### Функции

```typescript
const moveToFront = (id: string) => {
  const index = elements.findIndex(el => el.id === id)
  if (index === -1 || index === elements.length - 1) return
  
  const newElements = [...elements]
  const [element] = newElements.splice(index, 1)
  newElements.push(element)
  
  updateScene(newElements)
}

const moveForward = (id: string) => {
  const index = elements.findIndex(el => el.id === id)
  if (index === -1 || index === elements.length - 1) return
  
  const newElements = [...elements]
  ;[newElements[index], newElements[index + 1]] = 
   [newElements[index + 1], newElements[index]]
  
  updateScene(newElements)
}

const moveBackward = (id: string) => {
  const index = elements.findIndex(el => el.id === id)
  if (index === -1 || index === 0) return
  
  const newElements = [...elements]
  ;[newElements[index], newElements[index - 1]] = 
   [newElements[index - 1], newElements[index]]
  
  updateScene(newElements)
}

const moveToBack = (id: string) => {
  const index = elements.findIndex(el => el.id === id)
  if (index === -1 || index === 0) return
  
  const newElements = [...elements]
  const [element] = newElements.splice(index, 1)
  newElements.unshift(element)
  
  updateScene(newElements)
}
```

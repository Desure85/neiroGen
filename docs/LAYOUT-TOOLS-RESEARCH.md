# 🔬 Исследование инструментов для создания макетов

**Дата:** 5 ноября 2025  
**Цель:** Определить необходимые инструменты для логопедов и нейропсихологов при создании макетов упражнений

---

## 📋 Типы упражнений и требуемые инструменты

### 1. 🗣️ Логопедические упражнения

#### Артикуляционная гимнастика
**Нужны:**
- Изображения положений языка/губ
- Стрелки для показа движений
- Нумерация последовательности действий
- Текстовые инструкции

#### Постановка звуков
**Нужны:**
- Слоговые таблицы (СА-СО-СУ-СЫ)
- Выделение звуков цветом
- Позиции артикуляции
- Транскрипция [р], [л], [ш]

#### Дифференциация звуков
**Нужны:**
- Таблицы для сравнения (С-Ш, Р-Л)
- Поля для сортировки слов
- Цветовое кодирование
- Линии соединения

### 2. 🧠 Нейропсихологические упражнения

#### Графические диктанты
**Нужны:**
- Сетка с клетками
- Стартовая точка
- Стрелки направлений
- Числовые метки

#### Зрительно-моторная координация
**Нужны:**
- Линии (прямые, волнистые, зигзаг)
- Точки для соединения
- Лабиринты
- Обводки по контуру

#### Внимание и память
**Нужны:**
- Геометрические фигуры
- Таблицы Шульте
- Области для группировки
- Чекбоксы/маркеры

#### Пространственное мышление
**Нужны:**
- Стрелки (вверх, вниз, влево, вправо)
- Координатные сетки
- Формы с поворотом
- Зеркальное отображение

## 🛠️ Рекомендуемые инструменты для макета

### Базовые элементы (уже есть)
- ✅ Текст
- ✅ Изображение
- ✅ Инструкции
- ✅ Сетка

### Графические элементы (добавить)
- 📐 **Линия** - для соединения, обводки, разделения
- ➡️ **Стрелка** - указание направления, последовательности
- ⭕ **Круг** - выделение, обводка объектов
- ▢ **Прямоугольник** - рамки, области
- ⭐ **Фигура** - треугольники, звезды для маркировки

### Интерактивные элементы (добавить)
- ✏️ **Поле ввода** - для записи ответов
- ☐ **Чекбокс** - для выбора/отметки
- 🎯 **Точка** - для соединения линиями
- 📝 **Область ответа** - пунктирная рамка для письма

### Специализированные (добавить)
- 🔢 **Номер** - автонумерация элементов
- 📊 **Таблица** - для слоговых таблиц, сравнения
- 🎨 **Цветовой маркер** - выделение цветом
- 📏 **Линейка/сетка** - для точных построений

## 💡 Приоритеты для первой итерации

### Must Have (критично)
1. **Линия** - базовый инструмент для большинства упражнений
2. **Стрелка** - показ направлений и последовательностей
3. **Круг/Прямоугольник** - выделение и группировка
4. **Номер** - автонумерация шагов

### Should Have (важно)
5. **Таблица** - слоговые таблицы и сравнения
6. **Поле ввода** - для письменных ответов
7. **Точка** - для упражнений "соедини"
8. **Цветовой маркер** - выделение важного

### Could Have (желательно)
9. **Фигуры** - треугольник, звезда, многоугольник
10. **Пунктирная линия** - для обводки
11. **Текстовая метка** - быстрая подпись
12. **Штамп** - часто используемые символы

## 🎯 Примеры использования

### Пример 1: Дифференциация С-Ш
```
[Таблица 2 колонки]
┌─────────┬─────────┐
│    С    │    Ш    │
├─────────┼─────────┤
│ [____]  │ [____]  │
│ [____]  │ [____]  │
└─────────┴─────────┘

Слова для сортировки:
САНКИ → [линия] → колонка С
ШАПКА → [линия] → колонка Ш
```

### Пример 2: Последовательность действий
```
[Номер 1] → [Стрелка] → [Номер 2] → [Стрелка] → [Номер 3]
  [Текст]                [Текст]                [Текст]
  "Вдох"               "Задержка"             "Выдох"
```

### Пример 3: Графический диктант
```
[Сетка 10x10]
[Точка старта] → [Стрелка вправо] → 3 клетки
                 [Стрелка вниз] → 2 клетки
                 [Стрелка влево] → 3 клетки
```

### Пример 4: Соедини картинки
```
[Изображение 1] ○─────────○ [Изображение A]
[Изображение 2] ○         ○ [Изображение B]
[Изображение 3] ○─────────○ [Изображение C]
```

# Элементы макета упражнения (WorksheetLayoutEditor)

## Обзор

Canvas-редактор макета теперь поддерживает элементы, специфичные для упражнений. Редактор автоматически получает данные о типе упражнения, его параметрах и инструкциях.

## Типы элементов

### Базовые элементы

#### 1. **Текст** (`text`)
- Текстовый блок с настройкой шрифта и цвета
- Параметры: `text`, `fontSize`, `fill`

#### 2. **Изображение** (`image`)
- Изображение по URL
- Параметры: `url`, `opacity`

#### 3. **Placeholder** (`placeholder`)
- Область-заглушка для будущего контента
- Параметры: `name`, `fill`, `stroke`

---

### Элементы упражнений ✨ NEW

#### 4. **Инструкции** (`instructions`)
- Блок с инструкциями к упражнению
- Автоматически подтягивает `draft.instructions` из конструктора
- Параметры: 
  - `instructionsList?: string[]` — список инструкций
  - `showNumbers?: boolean` — нумеровать ли инструкции

#### 5. **Игровое поле** (`exercise_field`)
- Область для размещения игрового поля упражнения
- Автоматически получает данные упражнения (`exerciseData`)
- Универсальный элемент для всех типов упражнений

#### 6. **Область ответов** (`answer_area`)
- Зона для ввода/записи ответов пациентом
- Визуально выделена цветом (голубой фон, синяя рамка)

#### 7. **Сетка** (`grid`) 
- Специфичный элемент для графических диктантов
- Отображается только когда `exerciseType === 'graphic_dictation'`
- Параметры:
  - `gridSize?: { width: number; height: number }` — размер сетки
  - `cellSize?: number` — размер ячейки
  - `exerciseData?: any` — данные графического диктанта

#### 8. **Пример** (`example`)
- Пример выполнения упражнения
- Показывает образец правильного ответа

---

## Использование

### В конструкторе упражнений

```tsx
<WorksheetLayoutEditor 
  value={draft.layout} 
  onChange={handleLayoutChange}
  exerciseType={draft.type}              // Тип упражнения
  exerciseData={graphicDictationResult || customParams}  // Данные упражнения
  instructions={draft.instructions}       // Инструкции
/>
```

### Автоматическая привязка данных

Элементы упражнений автоматически связываются с данными:

1. **`instructions`** → получает `draft.instructions` из конструктора
2. **`grid`**, **`exercise_field`** → получает `exerciseData` (например, данные графического диктанта)
3. Кнопка **"Сетка"** появляется только для `graphic_dictation`

## Структура данных

### CanvasElement (расширенный)

```typescript
export interface CanvasElement {
  // ... базовые поля (id, type, x, y, width, height, rotation)
  
  // Элементы упражнений
  gridSize?: { width: number; height: number }  // Для grid
  cellSize?: number                              // Для grid
  showNumbers?: boolean                          // Для instructions
  instructionsList?: string[]                    // Для instructions
  exerciseData?: any                             // Данные упражнения
}
```

### Доступные типы элементов

```typescript
export type CanvasElementType = 
  // Базовые
  | 'text' 
  | 'image' 
  | 'placeholder'
  // Элементы упражнений
  | 'instructions'         
  | 'grid'                 
  | 'exercise_field'       
  | 'answer_area'          
  | 'example'
```

## Контекстные элементы для типов упражнений

Элементы макета **автоматически фильтруются** в зависимости от типа упражнения:

### Для **графического диктанта** (`graphic_dictation`)
Доступны:
- ✅ Базовые: текст, изображение, placeholder
- ✅ **Инструкции** — автоматически подтягивают команды диктанта
- ✅ **Сетка** — с настройками размера и ячеек
- ❌ Игровое поле — не нужно
- ❌ Область ответов — не нужно  
- ❌ Пример — не нужно

### Для других типов упражнений
Доступны:
- ✅ Базовые элементы
- ✅ Инструкции
- ✅ Игровое поле
- ✅ Область ответов
- ✅ Пример
- ❌ Сетка — только для графического диктанта

---

## Преимущества

✅ **Автоматизация** — элементы подтягивают данные из упражнения  
✅ **Типизация** — каждый тип упражнения имеет свой набор элементов  
✅ **Контекстность** — показываются только релевантные элементы  
✅ **Чистота UI** — нет лишних кнопок для данного типа

## Визуальный рендеринг элементов ✅

Все элементы упражнений рендерятся с уникальным дизайном:

### 📝 Инструкции
- Голубой фон (#eff6ff)
- Синяя рамка
- Заголовок "📝 ИНСТРУКЦИИ"
- Нумерованный список или маркеры
- Автоматически подтягивает `draft.instructions`

### 📐 Сетка
- Белый фон с серой рамкой
- Реальная сетка с линиями
- Размер сетки указан внизу (например, "📐 Сетка 16×16")
- Динамический размер через `gridSize` и `cellSize`

### 🎮 Игровое поле
- Зелёный фон (#f0fdf4)
- Толстая зелёная рамка
- Иконка 🎮
- Показывает тип упражнения

### ✍️ Область ответов
- Голубой фон (#dbeafe)
- Синяя пунктирная рамка
- Подсказка "Пациент записывает/рисует здесь"

### 💡 Пример
- Жёлтый фон (#fef9c3)
- Жёлтая рамка
- Иконка 💡 и заголовок "ПРИМЕР"
- Описание назначения

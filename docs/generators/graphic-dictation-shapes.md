# Графическое диктование - Библиотека фигур

## Доступные фигуры

### Простые (easy) - 4-15 шагов
- **square** - Квадрат
- **rectangle** - Прямоугольник  
- **house** - Домик (квадрат + треугольная крыша)
- **tree** - Дерево/ёлка

### Средние (medium) - 12-40 шагов
- **car** - Машинка
- **boat** - Лодка/кораблик
- **robot** - Робот
- **airplane** - Самолёт
- **rocket** - Ракета
- **butterfly** - Бабочка

### Сложные (hard) - 30-70 шагов
- **dog** - Собачка
- **cat** - Кошка
- **mixer_truck** - Бетономешалка

## Два режима генерации

### 1. Из шаблонов (рекомендуется)
14 готовых фигур, идеальное качество:
- Просто укажите название: "домик", "кран", "ракета"
- Мгновенная генерация
- Гарантированно узнаваемые фигуры

### 2. Из изображения (для своих фигур)
Загрузите любое изображение:
- Силуэты и иконки
- Полный контроль через параметры
- Точная трассировка контура

Подробнее: [Режим "Из изображения"](./graphic-dictation-image-mode.md)

**LLM интеграция удалена** - показала низкое качество для точных фигур.

## Примеры запросов API

### Генерация домика (из шаблона)
```json
{
  "description": "домик",
  "difficulty": "easy",
  "grid_width": 16,
  "grid_height": 16,
  "cell_size_mm": 10,
  "allow_diagonals": false,
  "shards": 1
}
```

### Генерация бетономешалки (из шаблона - сложный)
```json
{
  "description": "бетономешалка",
  "difficulty": "hard",
  "grid_width": 25,
  "grid_height": 25,
  "cell_size_mm": 10,
  "allow_diagonals": false,
  "shards": 1
}
```

### Генерация из изображения (новое!)
```json
{
  "source_image": "https://example.com/crane-icon.png",
  "grid_width": 20,
  "grid_height": 20,
  "cell_size_mm": 10,
  "simplification": 1.5,
  "smoothing": 1,
  "allow_diagonals": false,
  "shards": 1
}
```
✅ **Преимущества:** Точность, полный контроль, любые фигуры  
⏱️ **Время генерации:** 1-3 секунды

### Случайная фигура среднего уровня

## Режим работы генератора

### Template Mode (рекомендуется)
- Укажите `description` или `shape_name`
- Генератор выберет подходящий шаблон или создаст фигуру через AI
- Гарантирует читаемые инструкции (10-70 шагов)

### Image Mode (legacy)
- Укажите `source_image` (URL или base64)
- Использует векторизацию изображения
- Может давать много шагов (не рекомендуется для обучения детей)

## Тестирование через UI

1. Откройте генератор графического диктанта
2. Выберите **"По описанию"**
3. Введите название фигуры:
   - `домик` - простая фигура
   - `ракета` - средняя сложность
   - `бетономешалка` - сложная фигура
4. Выберите уровень сложности
5. Настройте размер сетки (рекомендуется 15-25 для детей)
6. Нажмите "Сгенерировать"

## Результат

Генератор вернёт:
- **commands**: Список команд (direction, steps)
- **instructions**: Человеко-читаемые инструкции на русском
- **preview_image_url**: PNG превью фигуры на сетке
- **preview_svg_url**: SVG векторное превью
- **start_row, start_col**: Начальная позиция

## Расширение библиотеки

Чтобы добавить новую фигуру:

1. Откройте `/app/server/generators/graphicdictation/internal/shapes/templates.go`
2. Добавьте запись в `GetTemplates()`:
```go
{
    Name:       "star",
    Category:   "basic",
    Difficulty: "medium",
    MinSteps:   10,
    MaxSteps:   20,
    Generator:  generateStar,
},
```
3. Реализуйте функцию `generateStar(width, height int, rng *rand.Rand) []image.Point`
4. Добавьте ключевые слова в `/internal/shapes/matcher.go`
5. Пересоберите и перезапустите воркер

## 1. Groq (рекомендуется!) 🚀

**Почему Groq:**
- ✅ Совершенно **бесплатно** (даже без кредитной карты)
- ✅ **Очень быстрая** генерация (2-5 секунд)
- ✅ Модель **Llama 3.3 70B** - новейшая, отличное качество
- ✅ Щедрые лимиты: 6000 токенов/минуту
- ✅ OpenAI-совместимый API

### Настройка Groq

**Шаг 1: Получить API ключ**
1. Зайдите на https://console.groq.com
2. Зарегистрируйтесь (email, без карты)
3. Перейдите в **API Keys**: https://console.groq.com/keys
4. Нажмите **Create API Key**
5. Скопируйте ключ (начинается с `gsk_...`)

**Шаг 2: Настроить проект**
Создайте/отредактируйте `.env`:
```bash
LLM_PROVIDER=groq
GROQ_API_KEY=gsk_ваш_ключ_здесь
```

**Шаг 3: Перезапустить воркер**
```bash
docker-compose restart go-graphic-dictation-worker
```

**Шаг 4: Проверить**
```bash
docker logs neirogen_go_gd_worker --tail 20
# Не должно быть ошибок "LLM API key not configured"
```

**Готово!** Теперь можете генерировать любые фигуры: "подъёмный кран", "замок", "слон" и т.д.

## 2. Google Gemini (тоже отлично!) 💎

**Почему Gemini:**
- ✅ **Бесплатно** до 60 запросов/минуту
- ✅ **Gemini 1.5 Flash** - быстрая и качественная модель
- ✅ От Google - надёжность и стабильность
- ✅ Больше токенов на запрос

### Настройка Gemini

**Шаг 1: Получить API ключ**
1. Зайдите на https://makersuite.google.com/app/apikey
2. Войдите через Google аккаунт
3. Нажмите **Create API Key**
4. Выберите Google Cloud проект (или создайте новый)
5. Скопируйте ключ (начинается с `AIza...`)

**Шаг 2: Настроить проект**
```bash
LLM_PROVIDER=gemini
GEMINI_API_KEY=AIza_ваш_ключ_здесь
```

**Шаг 3: Перезапустить**
```bash
docker-compose restart go-graphic-dictation-worker
```

**Лимиты:**
- Бесплатный tier: 60 запросов/минуту
- 1500 запросов/день
- Отлично для разработки и небольших проектов

## 3. Ollama (локально, оффлайн) 🏠

**Почему Ollama:**
- ✅ Полностью **бесплатно** и **локально**
- ✅ Работает **без интернета**
- ✅ Полная **конфиденциальность**
- ✅ Без лимитов запросов
- ❌ Медленнее (зависит от железа)
- ❌ Качество ниже чем GPT-4/Gemini

### Настройка Ollama

**Шаг 1: Установить Ollama**

**Linux:**
```bash
curl -fsSL https://ollama.ai/install.sh | sh
```

**macOS:**
```bash
brew install ollama
```

**Windows:**
Скачайте с https://ollama.ai/download

**Шаг 2: Запустить модель**
```bash
# Запустить llama 3.1 (8B - быстрая, 70B - качественная)
ollama run llama3.1:8b

# Или более качественная модель (нужно больше RAM)
ollama run llama3.1:70b
```

**Шаг 3: Настроить проект**

В `.env`:
```bash
LLM_PROVIDER=ollama
LLM_API_URL=http://host.docker.internal:11434/v1/chat/completions
LLM_MODEL=llama3.1:8b
GROQ_API_KEY=dummy
```

> **Примечание:** `GROQ_API_KEY=dummy` нужен чтобы проходила проверка IsAvailable()

**Для Linux (Docker):**
Если `host.docker.internal` не работает:
```bash
LLM_API_URL=http://172.17.0.1:11434/v1/chat/completions
```

**Шаг 4: Перезапустить**
```bash
docker-compose restart go-graphic-dictation-worker
```

**Требования к железу:**
- **llama3.1:8b** - минимум 8GB RAM
- **llama3.1:70b** - минимум 64GB RAM
- GPU ускоряет в 5-10 раз

## Troubleshooting

### Groq: "rate limit exceeded"
Вы превысили 6000 токенов/минуту. Подождите минуту или зарегистрируйте второй аккаунт.

### Gemini: "API key not valid"
- Убедитесь что включили Gemini API в Google Cloud Console
- Ключ должен начинаться с `AIza`
- Может потребоваться активация billing (но платить не нужно для free tier)

### Ollama: "connection refused"
- Проверьте что Ollama запущена: `ollama list`
- Для Linux Docker: используйте `http://172.17.0.1:11434`
- Проверьте firewall

### Все провайдеры: "LLM API key not configured"
```bash
# Проверьте переменные окружения в контейнере
docker exec neirogen_go_gd_worker env | grep -E "LLM|GROQ|GEMINI"
```

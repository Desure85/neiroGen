# Автономный IDE-агент для разработки приложения генерации нейрологопедических и логопедических заданий  

## Стек технологий
- **Backend (основной):** PHP 8.3+ (Laravel или Symfony)  
- **Runtime и брокер:** RoadRunner (Go)  
- **High-performance сервисы:** Go (плагины или микросервисы для TTS, WebSocket, real-time)  
- **UI:** React + Next.js (SSR/SPA), далее возможен React Native для мобильных клиентов  
- **БД:** PostgreSQL (jsonb для упражнений и заданий), Redis (кэш/очереди)  
- **Тестирование:** PHPUnit + Pest, phpstan, phpcs, jest/react-testing-library  
- **Docker** Все размещаем в докер контейнерах, на машине ничего не устанавливаем

## Принципы работы агента
1. Всегда сначала подробно анализируй задачу.  
2. Делай пошаговый разбор: цель → план → риски → реализация → проверка.  
3. Следи за соответствием PSR-12, строгая типизация PHP, phpdoc, nullable-аргументы, edge-cases.  
4. Если код изменяется — объясняй **почему именно так** и какие альтернативы рассматривались.  
5. Проверяй результат по цепочке: линтер → статический анализ → тесты → ручной прогон логики.  
6. Работай до полного завершения задачи.  

## Алгоритм выполнения задач
- Определи цель и возможные подзадачи.  
- Составь детализированный план (с несколькими вариантами решений).  
- Выбери оптимальный вариант.  
- Реализуй решение: редактируй/создавай файлы, выполняй команды (`composer`, `rr`, `phpunit`, `phpstan`, `phpcs`, `artisan`/`console`, `npm`, `jest`).  
- Запусти проверки и проанализируй результат.  
- При ошибках или недочётах исправляй до идеала.  
- Представь финальный код и краткий отчёт:  
  - что сделано,  
  - какие улучшения,  
  - почему выбранное решение оптимально.

## Текущая задача
Создать генератор заданий (логопедических, нейрологопедических, нейропсихологических), которые выводятся:  
- на **листах A4** (PDF, 2–3 задания на страницу),  
- в **интерактивном UI** (React/Next.js),  
- с разделением по категориям.  

## Категории заданий

###Логопедические / Нейрологопедические
 - Фонематический слух (различение звуков).
 - Артикуляционные упражнения (с картинками).
 - Подбор слов к картинкам.
 - Составление предложений.
 - Найди лишнее слово/предмет.
 - Игры «больше–меньше–средний», «быстрее–медленнее».

###Нейропсихологические
 - Память → запомни картинки, лишнее слово, ассоциации, последовательность действий.
 - Внимание → таблицы Шульте, буквы/фигуры, «найди ошибки», линии-лабиринты.
 - Пространственные функции → зеркальные фигуры, танграм, лабиринты, ориентация «право–лево».
 - Графомоторика → обводка по точкам, штриховка, соединение предметов, узоры, копирование фигур.
 - Речь → составь рассказ по картинкам, опиши нелепицу, подбери противоположности.
 - Логика → последовательности, причинно-следственные связи, группировка, числовые ряды.

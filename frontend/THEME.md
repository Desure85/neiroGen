# 🎨 Дизайн-система NeuroGen

## Выбранная тема: Shadcn/ui + Custom Medical Theme

### Основные принципы дизайна:

#### 🌟 **Для детей:**
- Яркие, но не отвлекающие цвета (синий + зеленый)
- Крупные кнопки с градиентами
- Мягкие анимации (bounce, pulse)
- Иконки и эмодзи для вовлеченности
- Закругленные углы и тени

#### 👨‍⚕️ **Для логопедов:**
- Чистый, профессиональный интерфейс
- Хорошая контрастность для читаемости
- Структурированная информация
- Легкая навигация

### Цветовая палитра:

```css
/* Медицинские цвета */
--medical-primary: 217 91% 60%    /* Синий */
--medical-secondary: 142 76% 36%  /* Зеленый */
--medical-accent: 45 93% 58%       /* Желтый */
--medical-success: 142 76% 36%    /* Успех */
--medical-warning: 45 93% 58%     /* Предупреждение */
--medical-danger: 0 84% 60%       /* Ошибка */
--medical-info: 199 89% 48%       /* Информация */

/* Детские цвета */
--child-primary: 217 91% 60%      /* Синий */
--child-secondary: 142 76% 36%    /* Зеленый */
--child-accent: 45 93% 58%        /* Желтый */
```

### Компоненты:

#### 1. **ThemeProvider** - Поддержка темной/светлой темы
```tsx
<ThemeProvider
  attribute="class"
  defaultTheme="light"
  enableSystem
  disableTransitionOnChange
>
```

#### 2. **ThemeToggle** - Переключатель темы
- Солнце/луна иконки
- Плавная анимация
- Доступность (screen reader)

#### 3. **Custom CSS Classes**:
- `.child-exercise` - Стили для упражнений
- `.child-button` - Детские кнопки
- `.child-text` - Детский текст
- `.medical-card` - Медицинские карточки
- `.medical-button-*` - Медицинские кнопки

### Анимации:

#### Для детей:
- `animate-bounce-gentle` - Мягкое подпрыгивание
- `animate-pulse-soft` - Мягкое пульсирование
- `hover:scale-105` - Увеличение при наведении

#### Для интерфейса:
- `transition-all duration-200` - Плавные переходы
- `transform hover:scale-105` - Эффекты при наведении

### Доступность (Accessibility):

✅ **WCAG 2.1 AA compliant**
✅ **Keyboard navigation**
✅ **Screen reader support**
✅ **High contrast ratios**
✅ **Focus indicators**
✅ **Semantic HTML**

### Рекомендации по использованию:

1. **Для детских упражнений:**
   ```tsx
   <div className="child-exercise">
     <button className="child-button">
       Начать упражнение
     </button>
   </div>
   ```

2. **Для интерфейса логопеда:**
   ```tsx
   <div className="medical-card">
     <button className="medical-button-primary">
       Сохранить прогресс
     </button>
   </div>
   ```

3. **Тема:**
   - Светлая тема по умолчанию
   - Темная тема для работы в помещении
   - Автоматическое переключение по системной теме

### Преимущества выбранного подхода:

✅ **Современный** - Shadcn/ui актуальный и поддерживаемый
✅ **Доступный** - Встроенная поддержка accessibility
✅ **Кастомизируемый** - Легко адаптировать под нужды
✅ **Производительный** - Минималистичный CSS
✅ **Консистентный** - Единая дизайн-система
✅ **Масштабируемый** - Легко добавлять компоненты

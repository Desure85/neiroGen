## 🚨 Основные проблемы

### 1. **Ошибка аутентификации**
Все тесты, требующие входа в систему, проваливаются с ошибкой:
```
Expected pattern: /\/(therapist|admin)/
Received string: "http://127.0.0.1:3001/login"
```

**Причина:** Бэкенд сервер не запущен или недоступен на порту 8000.

### 2. **Отсутствие бэкенда**
Тесты пытаются обратиться к API эндпоинтам, но получают ошибки подключения.

## 🔧 Решение проблем

### 1. **Запуск бэкенда**
```bash
# Запустить Laravel приложение
cd h:\work\neirogen\neiroGen\app
php artisan serve --host=0.0.0.0 --port=8000

# Или через RoadRunner
./rr serve
```

### 2. **Настройка переменных окружения**
```bash
# В директории frontend
export BASE_URL=http://127.0.0.1:3001
export E2E_ADMIN_EMAIL=admin@demo.local
export E2E_ADMIN_PASS=password
```

### 3. **Запуск frontend сервера**
```bash
cd h:\work\neirogen\neiroGen\frontend
npm run dev
```

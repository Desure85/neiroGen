## 📐 Архитектура

### Типы элементов

```typescript
type CanvasElementType =
  | 'line'      // ← Объединяет линию и стрелку
  | 'shape'     // ← Объединяет все фигуры
  | 'number'
  | 'list'
  | 'table'
  | 'checkbox'
```

### Свойства

```typescript
interface CanvasElement {
  // Базовые
  x, y, width, height
  
  // Для line:
  x2, y2                    // Конечная точка
  lineStyle:                // Тип линии
    | 'solid'               // Сплошная —
    | 'dashed'              // Пунктир - - -
    | 'dotted'              // Точки · · ·
    | 'arrow-end'           // Стрелка →
    | 'arrow-both'          // Двунаправленная ⟷
    | 'arrow-dot'           // С точкой на старте •→
  
  // Для shape:
  shapeType:                // Тип фигуры
    | 'rectangle'           // Прямоугольник
    | 'circle'              // Круг
    | 'ellipse'             // Эллипс
    | 'triangle'            // Треугольник
    | 'star'                // Звезда
    | 'hexagon'             // Шестиугольник
  cornerRadius: number      // Скругление углов (для прямоугольника)
}
```

## 💡 Преимущества

### 1. Простота добавления новых типов

**Было:** Добавить кнопку → реализовать логику → обновить панель свойств  
**Стало:** Добавить в dropdown → готово!

```typescript
// Добавить новый тип стрелки:
lineStyle: 'arrow-end' | 'arrow-both' | 'arrow-dot' | 'arrow-circle' ← просто добавь!
```

### 2. Меньше загромождения UI

**Было:**
```
[— Линия] [→ Стрелка] [○ Круг] [□ Прямоугольник] 
[⬭ Эллипс] [▲ Треугольник] [★ Звезда] ...
```

**Стало:**
```
[— Линия] [○ Фигура] [# Номер] ...
```

### 3. Единообразное растягивание

**Все фигуры** теперь растягиваются одинаково:
- Круг → изменяется width/height → становится эллипсом!
- Прямоугольник → изменяется width/height
- Треугольник → изменяется width/height

### 4. Расширяемость

Легко добавить:
- Линию: `'arrow-dashed'`, `'double'`, `'wavy'`
- Фигуру: `'pentagon'`, `'octagon'`, `'heart'`, `'custom'`

## 📦 Реализация

### Создание элемента

```typescript
// Создать линию
addElement('line')
// → lineStyle: 'solid' (по умолчанию)

// Создать фигуру  
addElement('shape', 'rectangle')
// → shapeType: 'rectangle'
```

### Изменение типа в UI

```typescript
// Пользователь выбрал "Стрелка →" в dropdown
updateElement(id, { lineStyle: 'arrow-end' })

// Пользователь выбрал "Круг" в dropdown
updateElement(id, { shapeType: 'circle' })
```

### Рендеринг

```typescript
// Линия
if (element.type === 'line') {
  const points = [element.x, element.y, element.x2, element.y2]
  
  if (element.lineStyle === 'arrow-end') {
    return <KonvaArrow points={points} pointerLength={10} />
  } else if (element.lineStyle === 'dashed') {
    return <KonvaLine points={points} dash={[10, 5]} />
  } else {
    return <KonvaLine points={points} />
  }
}

// Фигура
if (element.type === 'shape') {
  if (element.shapeType === 'circle') {
    const radius = Math.min(element.width, element.height) / 2
    return <KonvaCircle radius={radius} />
  } else if (element.shapeType === 'rectangle') {
    return <KonvaRect 
      width={element.width} 
      height={element.height}
      cornerRadius={element.cornerRadius}
    />
  } else if (element.shapeType === 'ellipse') {
    return <KonvaEllipse 
      radiusX={element.width / 2}
      radiusY={element.height / 2}
    />
  }
}
```

## 🚀 Возможности для специалистов

### Логопеды

**Соединение слогов:**
```
СА ──→ МА ──→ РА
```
**Как сделать:**
1. [— Линия] → добавить
2. В настройках выбрать "Стрелка →"
3. Растянуть между словами

**Выделение правильного ответа:**
```
САНКИ  ○
ШАНКИ  ○  ← обведи
ЗАНКИ  ○
```
**Как сделать:**
1. [○ Фигура] → добавить
2. В настройках выбрать "Круг"
3. Растянуть вокруг слова

### Нейропсихологи

**Последовательность действий:**
```
[1] ──→ [2] ──→ [3]
 ↓       ↓       ↓
```
**Как сделать:**
1. [# Номер] × 3
2. [— Линия] → в настройках "Стрелка →"
3. [— Линия] → в настройках "Стрелка →" (вниз)

**Сравнительная таблица:**
```
┌────────┬────────┐
│   С    │   Ш    │
├────────┼────────┤
│ САНКИ  │ ШАПКА  │
└────────┴────────┘
```
**Как сделать:**
1. [⊞ Таблица] → добавить
2. Настроить размеры
